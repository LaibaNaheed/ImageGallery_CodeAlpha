ImageGallery (Description)

This project is a responsive Image Gallery web application developed using HTML, CSS, and JavaScript. The gallery displays images in a structured grid layout and allows users to open images in an interactive lightbox view with navigation controls.

Users can filter images based on categories and smoothly browse through them using next and previous buttons. The design is fully responsive and optimized for both desktop and mobile devices.

This project was created as part of an internship task to strengthen frontend development skills and demonstrate practical implementation of DOM manipulation, event handling, and responsive UI design.

✨ Features

Responsive image grid layout

Category-based image filtering

Smooth hover effects with transitions

Lightbox view for enlarged images

Next and Previous image navigation

Close button for lightbox

Mobile-friendly design

Clean and modern lavender-themed interface

🛠 Technologies Used

HTML – Structure of the web page

CSS – Styling, layout, and responsive design

JavaScript – Functionality, filtering logic, and lightbox controls
